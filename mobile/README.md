# TeraVoo Mobile App

This project uses **Flutter**.

## Setup

1. Ensure you have Flutter SDK installed (`flutter doctor`).
2. Run in this directory:
   ```bash
   flutter create .
   ```
   *Note: This will scaffold the standard Flutter app structure into this folder.*

## Architecture (Planned)

- **State Management**: Provider or Riverpod.
- **Local DB**: SQLite (sqflite) for Offline-First.
- **HTTP**: Dio.

## Features (MVP)
- OTP Login
- Photo Capture & Upload
- Product List (Offline cached)
- Basic Chat
